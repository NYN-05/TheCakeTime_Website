### File Formats
- **Primary:** WebP (for modern browsers)

- **Logos:** SVG (scalable vector graphics)

### Quality Standards
- High quality, well-lit, professional looking
- 2x retina display support
- Balanced compression (70-85% JPEG quality)
- Color space: sRGB

## ğŸ¯ Optimization Guidelines

### Pre-Upload Checklist

#### Compression
- Use TinyPNG or ImageOptim for lossy compression
- Target 70-85% quality for JPEGs
- Convert PNGs to WebP where possible
- Product images: <200KB
- Hero images: <500KB

#### Responsive Versions (Optional)
Create 3 versions for optimal performance:
- Desktop: Full resolution (1920px width max)
- Tablet: Medium resolution (1024px width max)
- Mobile: Small resolution (640px width max)

#### File Naming
```
{category}-{type}-{variant}.{format}
Examples:
- product-birthday-chocolate.jpg
- hero-banner-wedding.jpg
- category-pastries.jpg
```

#### Image Metadata
- Title tag (descriptive, keyword-rich)
- Alt text (accessibility, SEO)
- Copyright information
