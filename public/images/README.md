## Image Guidelines

- **Format**: JPG, PNG, or WebP
- **Hero**: 1920x1080px or larger (landscape)
- **Categories**: 400x300px minimum (landscape)
- **Products**: 600x600px minimum (square)
- **Quality**: High quality, well-lit photos
